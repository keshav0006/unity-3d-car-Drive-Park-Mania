# Unity 3D Car Parking Simulation Game

A realistic 3D car parking simulation game built in Unity.
The game focuses on real-world parking challenges including
reverse and parallel parking with multiple playable levels.

## Gameplay Overview
- Multiple progressive parking levels
- Realistic car physics and controls
- Reverse and parallel parking challenges
- Game restarts and level progression logic

## Vehicle System
- 3 selectable car models
- Different handling behaviors
- Car selection before gameplay

## UI & Audio
- Fully functional UI system
- Level selection and feedback
- Audio integration for driving and interactions

## Tools & Technologies
- Unity (3D)
- C#
- Rigidbody & Wheel Colliders
- UI System
- Audio System
